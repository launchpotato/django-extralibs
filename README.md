# Django Extralibs
A collections of custom template tags and other library

## Installation

- Install from github

		pip install -e git://github.com/launchpotato/django-extralibs.git@0.1.1#egg=extralibs

- Add to your settings.py 

		INSTALLED_APPS = (
		    ...
		    'extralibs',
		    ...
		)
